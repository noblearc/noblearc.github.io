채널 운영에 필요한 전 과정을 자동화합니다.

00
딸각모드 — 전자동 원클릭 시스템
소재만 직접 입력하면, 그 외 모든 전 과정(원하면 업로드까지) 전부 원클릭 자동화 모드로 완성됩니다. 복잡한 설정 없이 버튼 하나로 끝냅니다.
01
AI 대본 자동 생성
단순 키워드 입력만으로 대본 생성, 떡상 영상 벤치마킹으로 대본 생성, 그 외 다양한 모드 지원.
영상 길이 · 다양한 카테고리 선택 가능(역사, 심리, 경제, 시사 등). 최상 품질 대본 자동 생성.
02
AI 이미지 자동 생성
대본에 맞는 이미지를 프로그램이 자동 생성. 쇼츠 14가지 · 롱폼 16가지 다양한 이미지 스타일 선택 가능.
03
AI 음성 합성 (TTS)
한국어, 일본어, 영어, 독일어, 스페인어 등 다양한 언어 지원 — 다국어 유튜브 채널 운영 가능. 다양한 목소리 · 고품질 TTS로 자동 음성 생성.
04
자동 자막 및 다양한 효과
BGM(배경음악), 화면 움직임 효과, 씬 전환 효과 등 다양한 효과 지원. 자막 자동 생성 및 정확한 싱크 맞춤.
05
씬 편집기
대본 · 이미지 간단하게 편집 가능. 이미지 대신 동영상 삽입도 가능. 100씬 이상 작업을 원활하게 지원합니다.
06
메타데이터 자동 생성
YouTube 제목 · 설명 · 태그를 AI가 자동으로 최적화합니다. 업로드 전 SEO 설정까지 원스톱 처리. 썸네일 이미지 생성도 지원합니다.
07
원클릭 렌더링 & YouTube 자동 업로드
설정 완료 후 버튼 한 번으로 완성된 MP4 파일이 출력됩니다. 롱폼(16:9)과 쇼츠(9:16) 각 프로그램별 지원. 렌더링 즉시 YouTube에 자동 업로드되는 기능도 지원합니다. 쇼츠는 대량 제작 큐(Queue) 기능으로 여러 영상을 연속 생성할 수 있습니다(대량 제작 한 번에 가능).
08
리서치 센터 — 경쟁자 분석 · 떡상 검색 · 뉴스 트렌드
채널명만 입력하면 경쟁 채널의 대시보드 · 업로드 패턴 · 최적 업로드 시간 · 수익 예측을 한눈에 분석합니다.
떡상 검색으로 최근 뜨고 있는 영상 · 이슈화되고 있는 영상을 발굴하여 주제 및 대본 벤치마킹 · 소재 발굴 지원.
뉴스 트렌드 검색으로 최근 뜨고 있는 검색어 · 이슈 뉴스 검색을 통해 다양한 소재 발굴을 지원합니다.

FAQ

자주 묻는 질문

어떤 AI 모델을 사용하나요?
Google Gemini, ElevenLabs / MiniMax / OpenAI TTS를 지원합니다.
API 비용은 어떻게 되나요?
NOBLE ARC 프로그램 요금과 별도로 AI API 실사용 비용이 발생합니다. 최대한 최소화된 비용으로 영상이 제작될 수 있도록 지원합니다. Gemini는 무료 티어로 시작할 수 있어 초기 비용이 거의 없습니다.
이용 기간이 끝나면 어떻게 되나요?
이용 기간이 만료되면 프로그램 사용이 중단됩니다. 재결제하면 즉시 다시 이용 가능합니다. 정기 자동 결제는 없으며, 필요할 때 직접 결제하는 방식입니다. 다양한 기간별 결제가 가능합니다.
Windows 외 Mac, Linux도 지원하나요?
현재는 Windows 전용으로 제공됩니다. Mac / Linux 버전은 추후 지원 예정입니다.
처음 시작할 때 어떤 준비가 필요한가요?
Google Gemini API 키만 있으면 바로 시작할 수 있습니다. Gemini는 무료 티어를 제공하므로 초기 비용 부담이 없습니다. 이후 고품질 TTS를 원하는 경우 ElevenLabs, MiniMax, OpenAI 등 추가 API 키를 선택적으로 등록할 수 있습니다.
라이선스 키를 잃어버렸어요.
회원 가입 후 로그인하면 마이페이지에서 구매한 라이선스 키를 언제든 다시 확인할 수 있습니다.
