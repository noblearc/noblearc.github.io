AI 대본 자동 생성
주제만 입력하면 Gemini AI가 완성도 높은 대본을 자동으로 작성합니다. 씬별 구성, 길이, 톤 모두 조절 가능합니다.
🎨
AI 이미지 자동 생성
대본의 내용과 맥락을 분석해 각 씬에 맞는 이미지를 자동 생성합니다. 다양한 아트 스타일 선택 가능.
🎙️
고품질 AI 음성 합성
Gemini, MiniMax, ElevenLabs, OpenAI 등 4가지 TTS 엔진 지원. 7개 언어, 다양한 목소리 선택 가능.
📝
정확한 자막 동기화
AI 음성인식(Whisper, Groq)으로 단어 단위 자막 싱크. 폰트, 크기, 색상, 위치 모두 커스터마이징 가능.
🎬
원클릭 영상 렌더링
FFmpeg 기반 고품질 렌더링. BGM 자동 삽입, 화면 전환 효과, 인트로/아웃트로 설정까지 모두 자동.
📺
유튜브 자동 업로드
완성된 영상을 유튜브에 바로 업로드. 제목, 설명, 태그, 썸네일까지 한 번에 설정 가능.

FAQ

어떤 PC 사양이 필요한가요?
Windows 10/11 64bit, RAM 16GB 이상 권장. 인터넷 연결 필수.
API 키가 별도로 필요한가요?
네, Gemini API 키(무료)가 기본으로 필요합니다. 고품질 TTS나 이미지 생성을 위해 추가 API 키를 사용할 수 있습니다.
구독료가 추가로 발생하나요?
프로그램 자체는 1회 결제로 영구 사용 가능합니다. 단, 사용하는 AI API 서비스의 요금은 별도입니다.
업데이트는 어떻게 받나요?
프로그램 실행 시 새 버전이 있으면 자동으로 알림이 뜨며, 클릭 한 번으로 업데이트됩니다. 업데이트는 무료입니다.
환불 정책은 어떻게 되나요?
구매 후 7일 이내, 정상 사용이 불가한 기술적 결함이 확인된 경우 전액 환불해드립니다.
