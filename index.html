채널 운영에 필요한 전 과정을 자동화합니다.

01
AI 대본 자동 생성
주제만 입력하면 Gemini AI가 완성도 높은 대본을 자동으로 작성합니다. 씬별 구성, 길이, 톤 조절 가능. 트렌드 리서치 기반 기획까지 한 번에.
02
AI 이미지 자동 생성
각 씬에 맞는 이미지를 Gemini가 자동 생성합니다. 다양한 아트 스타일 선택 가능.
03
AI 음성 합성 (TTS)
Gemini, ElevenLabs, MiniMax, OpenAI — 4가지 엔진을 지원합니다. 자연스러운 한국어 나레이션.
04
자동 자막 생성
Groq Whisper 기반 자막 자동 생성. 씬별 싱크 완벽 맞춤.
05
원클릭 렌더링
설정 완료 후 렌더링 버튼 한 번으로 완성된 MP4 파일을 출력합니다. 롱폼(16:9)과 쇼츠(9:16) 모두 지원. YouTube 자동 업로드까지 연결됩니다.

FAQ

자주 묻는 질문

어떤 AI 모델을 사용하나요?
Google Gemini (대본·이미지·TTS), Groq Whisper (자막), ElevenLabs / MiniMax / OpenAI TTS를 지원합니다. 각 서비스의 API 키만 있으면 바로 사용할 수 있습니다.
API 비용은 어떻게 되나요?
NOBLE ARC 프로그램 요금과 별도로, 사용하는 AI API(Gemini, ElevenLabs 등)의 실사용 비용이 발생합니다. Gemini는 무료 티어로 시작할 수 있어 초기 비용이 거의 없습니다.
구독 취소는 어떻게 하나요?
언제든 구독을 취소할 수 있습니다. 취소 후에는 해당 구독 기간이 끝날 때까지 정상 이용 가능합니다.
Windows 외 Mac, Linux도 지원하나요?
현재는 Windows 전용으로 제공됩니다. Mac / Linux 버전은 추후 지원 예정입니다.
영상 퀄리티는 어느 정도인가요?
최대 4K 해상도 MP4 출력을 지원합니다. 이미지 품질은 Gemini 이미지 생성 모델 기준이며, 외부 동영상 파일을 씬에 직접 삽입하는 것도 가능합니다.
